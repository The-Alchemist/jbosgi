I am happy to announce the release of JBossOSGi-2.0.0

You can download the binary here: jboss-osgi-installer-2.0.0.jar

This is the first R5 compliant release for WildFly-8.0.0 integration.

  • Compliant R5 Core Framework
  • Add support for execution environment [JavaSE-1.7]
  • Relicense repository code base to ASL2.0
  • Update Aries Blueprint to 1.0.0
  • Update Felix Configuration Admin to 1.6.0
  • Update Felix Declarative Services to 1.6.2
  • Update Felix EventAdmin to 1.3.0
  • Update Aries JMX to 1.0.0
  • Update Aries JNDI to 1.0.0

Please also have a look at the latest version of our User Guide.
